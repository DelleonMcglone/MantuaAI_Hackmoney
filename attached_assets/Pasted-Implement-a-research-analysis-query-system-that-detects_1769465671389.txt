Implement a research/analysis query system that detects analytical questions from users and outputs interactive charts with summaries and bullet points directly in the chat.

**Feature Overview:**
When a user asks a research question (price history, volume, pool analytics), display a rich response card containing:
- A relevant chart visualization
- A text summary of findings
- Bullet points with key insights

| Query Type | Example | Chart Type |

| Distribution | "My portfolio breakdown" | Pie/Donut chart |

**Query Detection Function:**

Create `src/utils/queryClassifier.ts`:
```ts
type QueryType = 'price' | 'volume' | 'comparison' | 'portfolio' | 'performance' | 'tvl' | 'action' | 'general';

interface ClassifiedQuery {
  type: QueryType;
  assets: string[];      // ["ETH", "mUSDC"]
  timeRange: string;     // "7d", "24h", "30d"
  chartType: 'line' | 'bar' | 'pie' | 'area';
  hook?: string;         // "nezlobin", "jit", "mev"
}

// Detection patterns:
// - Price: "price", "cost", "trading at", "worth"
// - Volume: "volume", "traded", "transactions", "activity"
// - Time: "yesterday", "last week", "days ago", "this month"
// - Comparison: "compare", "vs", "versus", "difference"
// - Hooks: "nezlobin", "jit", "mev protection", "directional fee"
```
